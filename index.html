Options:
auto - start flipping in automatic mode [true | false]
time - time before all cards flip [ms]
shadow - on/off shadow [true | false]
sequent - sequential time before neighboring cards flip [ms]
rotation-mode - simultaneous or sequential mode [simultaneous | sequential]
cards-per-row - Number of cards per row [number]
number-of-rows - Number of rows [number]

Features:
Auto stop on mouse over and on deactivated browser tab

Usage with pure JavaScript:


    <link href="css/flipping.css" rel="stylesheet"/>
    <script src="js/flipping.js"/>
    ...
    <div id="flippingcards" class="flipping">

        <div>
            <button>&#9668;</button>
        </div>


        <div class="deck">
            <div>
