Options:

autoflip-mode - start flipping in automatic mode [true | false]
autoflip-delay - delay before next flip in automatic mode [ms]
shadow - on/off shadow [true | false]
transition-duration - card flip transition duration [ms]
sequential-delay - sequential delay before neighboring cards flip [ms]
rotation-mode - simultaneous or sequential mode [simultaneous | sequential]
cards-per-row - Number of cards per row [number]
number-of-rows - Number of rows [number]
